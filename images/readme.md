# Moje Obrázky / My Images

Vložte svoje vlastné obrázky do tohto priečinka.
Put your own images into this folder.

### Ako to funguje:
1. Nahrajte obrázok koňa a pomenujte ho `kon.jpg` (alebo `kon.png`).
2. Kód v `HeroImageSection.tsx` ho automaticky načíta z cesty `./images/kon.jpg`.

### Odporúčané rozmery:
- Hero obrázok: 2000x850px (pomer 21:9)
- Moduly: 1000x800px
